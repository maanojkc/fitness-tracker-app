# 💪 Fitness Tracker Web App

A full-stack fitness tracking application built using Flask that allows users to monitor their body metrics, workout routines, and progress over time with interactive visualizations.

---

## 🚀 Features

- 🔐 User Authentication (Login & Register)
- 📊 Track Weight, Body Fat %, and Calories
- 🏋️ Workout Logging (Push / Pull / Legs / Rest)
- 📈 Progress Visualization using charts
- 🎯 Goal Tracking and Performance Monitoring
- 🗓️ Daily progress history tracking

---

## 🛠️ Tech Stack

- **Backend:** Python (Flask)
- **Frontend:** HTML, CSS
- **Database:** SQLite
- **Visualization:** Chart.js
